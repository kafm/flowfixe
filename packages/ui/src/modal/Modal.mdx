# Modal

Use Modal when you want to open a dialog with contextualized information that the user must act upon without interrupting the state of the main window.

<Unstyled>
  <ModalMainExample
    label="Open Modal"
    title="A Modal"
    contents="@Modal contents"
  />
</Unstyled>

```jsx
const Modal = () => {
  const [open, setOpen] = useState(false);
  return (
    <div>
      <Button onClick={() => setOpen(!open)} label="Open Modal" />
      <Modal open={open} title="A Modal" onClose={() => setOpen(false)}>
        @Modal Contents
        <Modal.Footer>
          <Button
            skin="primary"
            variant="outline"
            label="Close"
            onClick={() => setOpen(false)}
          ></Button>
        </Modal.Footer>
      </Modal>
    </div>
  );
};
```
