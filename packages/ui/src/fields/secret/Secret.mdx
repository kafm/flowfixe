# Secret field

The secret field allows the input of sensitive text.

<Unstyled>

<Secret name="secret" placeholder="Password" width="300px" />

<Secret name="secret" placeholder="Password" width="300px" toggleVisibility />

</Unstyled>

```jsx

<Secret name="secret" placeholder="Password" width="300px" />

<Secret name="secret" placeholder="Password" width="300px" toggleVisibility />

```

## Validation

Secret fields can have restrictions similar to the text field.

<Unstyled>
  <Secret
    name="password"
    label="Password"
    aid="Please enter your password"
    width="300px"
    toggleVisibility
    required
  />
</Unstyled>

```jsx
<Secret
  name="password"
  label="Password"
  aid="Please enter your password"
  width="300px"
  toggleVisibility
  required
/>
```

## Read only

Secret field defined as read-only cannot be modified.

<Unstyled>
  <Secret
    name="password"
    label="Password"
    aid="Please enter your password"
    value="Password"
    width="300px"
    toggleVisibility
    readOnly
    required
  />
</Unstyled>

```jsx
<Secret
  name="password"
  label="Password"
  aid="Please enter your password"
  width="300px"
  toggleVisibility
  readOnly
  required
/>
```
