# Select field

The select field allows users to select one option from a list of options.

<Unstyled>
  <Select
    name="lang"
    label="Programming Language"
    aid="Please select your favorite programming language"
    width="300px"
  >
    <SelectOption value="C/C++" />
    <SelectOption value="Go" />
    <SelectOption value="Java" />
    <SelectOption value="Typescript" />
    <SelectOption value="Javascript" />
  </Select>
</Unstyled>

```jsx
<Select
  name="lang"
  label="Programming Language"
  aid="Please select your favorite programming language"
>
  <SelectOption value="C/C++" />
  <SelectOption value="Go" />
  <SelectOption value="Java" />
  <SelectOption value="Typescript" />
  <SelectOption value="Javascript" />
</Select>
```

## Validation

Select field can have restrictions (e.g. required). Error messages will be displayed upon violation of the configured restrictions.

<Unstyled>
  <Select
    name="lang"
    label="Programming Language"
    placeholder="Select your favorite language"
    required
    width="300px"
  >
    <SelectOption value="C/C++" />
    <SelectOption value="Go" />
    <SelectOption value="Java" />
    <SelectOption value="Typescript" />
    <SelectOption value="Javascript" />
  </Select>
</Unstyled>

```jsx
<Select
  name="lang"
  label="Programming Language"
  placeholder="Select your favorite language"
  required
  width="300px"
>
  <SelectOption value="C/C++" />
  <SelectOption value="Go" />
  <SelectOption value="Java" />
  <SelectOption value="Typescript" />
  <SelectOption value="Javascript" />
</Select>
```

## Read only

Select field defined as read-only cannot be modified.

<Unstyled>
  <Select
    name="lang"
    label="Programming Language"
    placeholder="Select your favorite language"
    value="Javascript"
    readOnly
    width="300px"
  />
</Unstyled>

```jsx
<Select
  name="lang"
  label="Programming Language"
  placeholder="Select your favorite language"
  value="Javascript"
  readOnly
  width="300px"
>
  <SelectOption value="C/C++" />
  <SelectOption value="Go" />
  <SelectOption value="Java" />
  <SelectOption value="Typescript" />
  <SelectOption value="Javascript" />
</Select>
```
