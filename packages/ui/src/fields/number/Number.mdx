# Number field

A number field is an input that allows a user to enter a number.

<Unstyled>
<Number name="example1" placeholder="An example number field" width="300px" />
</Unstyled>

```jsx
<Number name="example1" placeholder="An example number field" />
```

## Validation

The field can have restrictions (e.g. required). Error messages will be displayed upon violation of the configured restrictions.

<Unstyled>
<Number
  name="quantity"
  label="Quantity"
  aid="Please enter an integer value between 10 and 1000"
  width="350px"
  min={10}
  max={1000}
  integer
  required
/>
</Unstyled>

```jsx
<Number
  name="quantity"
  label="Quantity"
  aid="Please enter an integer value between 10 and 1000"
  width="350px"
  min={10}
  max={1000}
  integer
  required
/>
```

## Read only

If defined as read-only, it cannot be modified.

<Unstyled>
<Number name="amount" value={100} readOnly width="300px" />
</Unstyled>

```jsx
<Number name="amount" value={100} value={100000.5} readOnly readOnly />
```

## Format

From decimal separator to prefix/suffix, you can define different formats for the number.

<Unstyled>
  <Number
    name="amount"
    value={10900.5}
    thousandsSeparator=","
    decimalSeparator="."
    prefix="$"
    width="300px"
  />

  <Number name="margin" value={40} suffix="%" width="300px" integer />
</Unstyled>

```jsx

<Number name="amount" value={10900.5} thousandsSeparator="," decimalSeparator="." prefix="$" />

<Number name="margin" value={40} suffix="%"  integer />

```
