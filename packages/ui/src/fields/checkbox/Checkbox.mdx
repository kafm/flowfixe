# Checkbox field

A checkbox field is an input that allows a user to use checkable interactive controls.

<Unstyled>
  <Checkbox name="example1" label="An example checkbox" />
</Unstyled>

```jsx
<Checkbox name="example1" label="An example checkbox" />
```

## Variants

Choose different variants to configure the checkbox's layout.

<Unstyled>

<Checkbox name="example2" label="An example checkbox" variant="default" />

<br />

<Checkbox name="example3" label="An example checkbox" variant="switch" />

</Unstyled>

```jsx

<Checkbox name="example2" label="An example square checkbox" variant="default"/>

<Checkbox name="example3" label="An example switch checkbox" variant="switch"/>

```

## Read only

Checkbox field defined as read-only cannot be modified.

<Unstyled>
  <Checkbox
    name="example4"
    label="An example read-only checkbox"
    aid="You cannot change this value"
    value={true}
    readOnly={true}
  />
</Unstyled>
