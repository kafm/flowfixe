# Date field

A date field is an input that allows a user to enter a date.

<Unstyled>
  <DatePicker
    name="example1"
    label="Date"
    aid="Specify a valid date"
    placeholder="DD-MM-YYYY"
    width="300px"
  />
</Unstyled>

```jsx
<DatePicker
  name="example1"
  label="Date"
  aid="Specify a valid date"
  placeholder="DD-MM-YYYY"
/>
```

## Validation

Date field can have restrictions (e.g. required). Error messages will be displayed upon violation of the configured restrictions.

<Unstyled>
  <DatePicker
    name="example2"
    label="Date"
    aid="Specify a valid date"
    placeholder="DD-MM-YYYY"
    required
  />
</Unstyled>

```jsx
<DatePicker
  name="example2"
  label="Date"
  aid="Specify a valid date"
  placeholder="DD-MM-YYYY"
  required
/>
```

## Read only

Date field defined as read-only cannot be modified.

<Unstyled>
  <DatePicker name="example3" label="Date" value={new Date()} readOnly />
</Unstyled>

```jsx
<DatePicker name="example3" label="Date" value={new Date()} readOnly />
```
