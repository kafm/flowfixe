# Textarea field

A textarea field is an input that represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text.

<Unstyled>
  <Textarea name="example1" placeholder="Type your multi-line text..." />
</Unstyled>

```jsx
<Textarea name="example1" placeholder="Type your multi-line text..." />
```

## Autogrow

By default, the textarea field doesnâ€™t change size based on the content it contains. You can set it to dynamically grow based on its content.

<Unstyled>
  <Textarea
    name="example1"
    placeholder="Type your multi-line text..."
    autoGrow
  />
</Unstyled>

```jsx
<Textarea name="example1" placeholder="Type your multi-line text..." autoGrow />
```

## Validation

Textarea field can have restrictions (e.g. required). Error messages will be displayed upon violation of the configured restrictions.

<Unstyled>
  <Textarea
    name="description"
    label="Description"
    aid="Please enter your description with a maximum length of 50 characters"
    maxLength={50}
    required
  />
</Unstyled>

```jsx
<Textarea
  name="description"
  label="Description"
  aid="Please enter your description with a maximum length of 50 characters"
  maxLength={50}
  required
/>
```

## Read only

Textarea field defined as read-only cannot be modified.

<Unstyled>

<Textarea
  name="example3"
  value="I expect to pass through this world but once;
Any good thing therefore that I can do, or any kindness that I can show to any fellow-creature, let me do it now;
Let me not defer or neglect it, for I shall not pass this way again."
  readOnly
  autoGrow
/>

</Unstyled>

```jsx
<Textarea
  name="example3"
  value="I expect to pass through this world but once;
```
