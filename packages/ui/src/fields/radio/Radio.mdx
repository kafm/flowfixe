# Radio field

A radio field allow the selection of a single option from a list of mutually exclusive options.

<Unstyled>
  <RadioGroup
    name="lang"
    label="Programming Language"
    aid="Please select your favorite programming language"
  >
    <Radio value="C/C++" />
    <Radio value="Go" />
    <Radio value="Javascript" />
    <Radio value="Typescript" />
  </RadioGroup>
</Unstyled>

```jsx
<RadioGroup
  name="lang"
  label="Programming Language"
  aid="Please select your favorite programming language"
>
  <Radio value="C/C++" />
  <Radio value="Go" />
  <Radio value="Javascript" />
  <Radio value="Typescript" />
</RadioGroup>
```

## Read only

Radio field defined as read-only cannot be modified.

<Unstyled>
  <RadioGroup
    name="lang2"
    label="Programming Language"
    aid="Please select your favorite programming language"
    value="Typescript"
    readOnly
  >
    <Radio value="C/C++" />
    <Radio value="Go" />
    <Radio value="Javascript" />
    <Radio value="Typescript" />
  </RadioGroup>
</Unstyled>

```jsx
<RadioGroup
  name="lang"
  label="Programming Language"
  aid="Please select your favorite programming language"
  value="Typescript"
  readOnly
>
  <Radio value="C/C++" />
  <Radio value="Go" />
  <Radio value="Javascript" />
  <Radio value="Typescript" />
</RadioGroup>
```
