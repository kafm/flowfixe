# Alert

Use alerts to provide feedback to your users.

## Skins

Choose different skins to configure the alert's color.

<Unstyled>

<Alert type="success" title="Success Alert!">
  <span>This is a success alert 😍</span>
</Alert>

<br />

<Alert type="info" title="Info Alert!">
  <span>This is a info alert 😎</span>
</Alert>

<br />

<Alert type="warning" title="Warning Alert!">
  <span>This is a warning alert 😦</span>
</Alert>

<br />

<Alert type="danger" title="Danger Alert!">
  <span>This is a danger alert 😨</span>
</Alert>

<br />

<Alert type="muted" title="Muted Alert!">
  <span>This is a muted alert 😋</span>
</Alert>

<Alert type="success" title="Success Alert!">

## Variants

Choose different variants to configure the alert's layout.

<Unstyled>
<Alert type="success" title="Success Alert!"  variant="normal" />

<br />

<Alert type="info" title="Info Alert!" variant="outline" />

<br />

<Alert type="muted" title="Muted Alert!" variant="text" />

</Unstyled>

```jsx

<Alert type="success" title="Success Alert!"  variant="normal" />

<Alert type="info" title="Info Alert!"  variant="outline" />

<Alert type="muted" title="Muted Alert!"  variant="text" />

```

## Alert with Actions

<Unstyled>
<Alert
  type="muted"
  title="We use cookies to improve your experience and for marketing."
>
  <Button label="Reject" size="sm" skin="light" />
  <Button label="Accept" size="sm" skin="primary" />
</Alert>

</Unstyled>

```jsx
<Alert
  type="muted"
  title="We use cookies to improve your experience and for marketing."
>
  <Button label="Reject" size="sm" skin="light" />
  <Button label="Accept" size="sm" skin="primary" />
</Alert>
```
