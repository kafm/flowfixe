# Icons

Flowfixe UI leverages the popular [React icons](https://react-icons.github.io/react-icons/) project, to seamless access to the entire Bootstrap Icons collection (1,500+ icons) as React components. 

We've enhanced the developer experience by removing the Bs prefix from imports and ensuring valid identifiers for all icons.

Here is an example of usage:

<HandThumbsUp style={{ fontSize: '16px', color: '#08c' }} />

```jsx
import { HandThumbsUp } from "@flowfixe/ui";
<HandThumbsUp style={{ fontSize: '16px', color: '#08c' }} />
```
