# Buttons

Use buttons to communicate actions that should occur upon clicks by the user. You can configure text labels, skins, icons, variants, and others to describe the action.


## Types and states

By default the button it's a clickable button. If you want to use the button to submit or reset a form's data, change it's type accordingly. You can make the button inactive by setting its state to the disabled. Loading state is also supported.

<Unstyled>
<div  style={{
    display: "flex",
    alignItems: "center",
    gap: "1rem",
    marginTop: "2rem",
  }}>
    <div>
        <Button skin="primary" type="submit" label="Submit Button" />
    </div>
    <div>
        <Button skin="default" variant="outline" type="reset" label="Reset Button" />
    </div>
    <div>
        <Button skin="primary" label="Disabled Button" disabled="true" />
    </div>
    <div>
        <Button skin="primary" label="Loading Button" loading="true" />
    </div>
</div>  
</Unstyled>

```jsx

<Button skin="primary" type="submit" label="Submit Button" />
  
<Button skin="default" variant="outline" type="reset" label="Reset Button" />
   
<Button skin="primary" label="Disabled Button" disabled />

<Button skin="primary" label="Loading Button" loading />

```

## Block level buttons

Stretch the button width to the width of its parent.

<Unstyled>
<div>
    <Button type="submit" label="Block level button" size="lg" stretch />
</div>
</Unstyled>

```jsx

<Button type="submit" label="Block level button" size="lg" stretch />

```
