# Dropdown

Use dropdown to display a contextualized menu of actions.

<Unstyled>
<Dropdown>
  <Dropdown.Trigger>
    <Button label="File" />
  </Dropdown.Trigger>
  <Dropdown.Menu>
    <Dropdown.Item label="New file..." />
    <Dropdown.Item label="New window" />
    <Dropdown.Item label="Open" />
  </Dropdown.Menu>
</Dropdown>
</Unstyled>

```jsx
<Dropdown>
  <Dropdown.Trigger>
    <Button label="File" />
  </Dropdown.Trigger>
  <Dropdown.Menu>
    <Dropdown.Item label="New file..." />
    <Dropdown.Item label="New window" />
    <Dropdown.Item label="Open" />
  </Dropdown.Menu>
</Dropdown>
```

## Icons

You can decorate your dropdown items with icons.

<Unstyled>
<Dropdown>
  <Dropdown.Trigger>
    <Button label="Dropdown with icons" />
  </Dropdown.Trigger>
  <Dropdown.Menu>
    <Dropdown.Item icon={<Heart />} label="Like" />
    <Dropdown.Item icon={<Star />} label="Star" />
  </Dropdown.Menu>
</Dropdown>
</Unstyled>

```jsx
<Dropdown>
  <Dropdown.Trigger>
    <Button label="Dropdown with icons" />
  </Dropdown.Trigger>
  <Dropdown.Menu>
    <Dropdown.Item icon={<Heart />} label="Like" />
    <Dropdown.Item icon={<Star />} label="Star" />
  </Dropdown.Menu>
</Dropdown>
```

## Positioning

## Shutcuts, keyboard controls and icons

You can add shortcuts, keyboard controls, and icons to the dropdown menu items.

<Unstyled>
<div
  style={{
    display: "flex",
    alignItems: "center",
    gap: "1rem",
    marginTop: "2rem",
  }}
>
  <div>
    <Dropdown keyboardControls>
      <Dropdown.Trigger>
        <Button label="Users" icon={<User/>} />
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Item icon={<UserPlus/>} label="Add user" />
        <Dropdown.Item icon={<UserXmark/>} label="Remove user" />
      </Dropdown.Menu>
    </Dropdown>
  </div>
  <div>
    <Dropdown keyboardControls>
      <Dropdown.Trigger>
        <Button label="Users" icon={<User/>} skin="light" />
      </Dropdown.Trigger>
      <Dropdown.Menu>
        <Dropdown.Item label="Add user" shortcut="⌘ N" />
        <Dropdown.Item label="Remove user" shortcut="⌘ ⌫" />
      </Dropdown.Menu>
    </Dropdown>
  </div>
</div>
</Unstyled>

```jsx

<Dropdown keyboardControls>
    <Dropdown.Trigger>
    <Button label="Users" icon={<User/>} />
    </Dropdown.Trigger>
    <Dropdown.Menu>
        <Dropdown.Item icon={<UserPlus/>} label="Add user" />
        <Dropdown.Item icon={<UserXmark/>} label="Remove user"/>
    </Dropdown.Menu>
</Dropdown>

<Dropdown keyboardControls>
    <Dropdown.Trigger>
```

## Properties

Dropdown properties:

<table>
  <tr>
    <td>
      <strong>Property</strong>
    </td>
    <td>
      <strong>Type</strong>
    </td>
    <td>
      <strong>Default</strong>
    </td>
    <td>
      <strong>Description</strong>
    </td>
  </tr>
  <tr>
    <td>placement</td>
    <td>string</td>
    <td>bottom</td>
    <td>
      Where to place the dropdown menu. Options are: <br />
      <code>top</code> <code>bottom</code><code>right</code><code>left</code>
    </td>
  </tr>
  <tr>
    <td>align</td>
    <td>string</td>
    <td>start</td>
    <td>
      How to align the dropdown menu. Options are: <br />
      <code>start</code> <code>end</code>
    </td>
  </tr>
</table>

Dropdown item properties:

label: string;
description?: string;
icon?: string;
shortcut?: string;
strong?: boolean;

<table>
  <tr>
    <td>
      <strong>Property</strong>
    </td>
    <td>
      <strong>Type</strong>
    </td>
    <td>
      <strong>Default</strong>
    </td>
    <td>
      <strong>Description</strong>
    </td>
  </tr>
  <tr>
    <td>label</td>
    <td>string</td>
    <td></td>
    <td>Label/text to display for the dropdown item</td>
  </tr>
  <tr>
    <td>icon</td>
    <td>string</td>
    <td></td>
    <td>The icon to display for the dropdown item</td>
  </tr>
  <tr>
    <td>shortcut</td>
    <td>string</td>
    <td></td>
    <td>The shortcut for performing the menu item action. </td>
  </tr>
  <tr>
    <td>strong</td>
    <td>boolean</td>
    <td>false</td>
    <td>When true the label of the menu item will be bold.</td>
  </tr>
</table>
