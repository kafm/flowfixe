# Avatar

Avatars are used to show a thumbnail representation of an individual or business in the interface.

<div
  style={{
    display: "flex",
    alignItems: "center",
    gap: "1rem",
    marginTop: "2rem",
  }}
>
    <div><Avatar src="https://i.pravatar.cc/150?u=a042581f4e29026704d" /></div>
</div>

```jsx

<Avatar src="https://i.pravatar.cc/150?u=a042581f4e29026704d" />

```

## Icons and placeholders

Icons  or text placeholders (e.g. name initials) can also be used.

<div
  style={{
    display: "flex",
    alignItems: "center",
    gap: "1rem",
    marginTop: "2rem",
  }}
>
    <div>
        <Avatar name="FF"  />
    </div>
    <div>
        <Avatar icon={<HandThumbsUp />} />
    </div>
</div>

```jsx
<Avatar name="FF"  />

<Avatar icon={<HandThumbsUp />} />
```

## Properties

Avatar properties:

<table>
    <tr>
        <td><strong>Property</strong></td>
        <td><strong>Type</strong></td>
        <td><strong>Default</strong></td>
        <td><strong>Description</strong></td>
    </tr>
    <tr>
        <td>name</td>
        <td>string</td>
        <td></td>
        <td>The name to use for the initials generation.</td>
    </tr>
    <tr>
        <td>src</td>
        <td>string</td>
        <td></td>
        <td>The url of the image to display in the avatar.</td>
    </tr>
    <tr>
        <td>icon</td>
        <td>string</td>
        <td></td>
        <td>The icon to use for the avatar.</td>
    </tr>
    <tr>
        <td>color</td>
        <td>string</td>
        <td>primary</td>
        <td>The color to use for the avatar. Options are: <br /><code>primary</code><code>dark</code><code>danger</code><code>warning</code><code>success</code><code>light</code><code>custom colors</code></td>
    </tr>
    <tr>
        <td>size</td>
        <td>string</td>
        <td>md</td>
        <td>The size of the avatar. Options are: <br /> <code>tiny</code><code>sm</code><code>md</code><code>lg</code></td>
    </tr>
    <tr>
        <td>tip</td>
        <td>string</td>
        <td></td>
        <td>Tooltip text to display for the avatar.</td>
    </tr>
    <tr>
        <td>disabled</td>
        <td>boolean</td>
        <td>false</td>
        <td>If true the avatar will be disabled.</td>
    </tr>
    <tr>
        <td>border</td>
        <td>boolean</td>
        <td>false</td>
        <td>If true the avatar will be bordered.</td>
    </tr>
    <tr>
        <td>borderColor</td>
        <td>string</td>
        <td>primary</td>
         <td>The color of the border. Options are: <br /><code>primary</code><code>dark</code><code>danger</code><code>warning</code><code>success</code><code>light</code><code>custom colors</code></td>
    </tr>
</table>
