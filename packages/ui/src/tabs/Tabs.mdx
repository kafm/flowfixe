# Tabs

Use tabs to display contents by sections where only one will be visible at a time. Upon clicking on a section/tab, body contents will change accordingly.

<Unstyled>
<Tabs>
  <Tab id="tab1" label="Music">
    @Tab1 contents
  </Tab>
  <Tab id="tab2" label="Videos">
    @Tab2 contents
  </Tab>
</Tabs>
</Unstyled>

```jsx

<Tabs>
  <Tab id="tab1" label="Music">
    @Tab1 contents
  </Tab>
  <Tab id="tab2" label="Videos">
    @Tab2 contents
  </Tab>
</Tabs>

```

## Icons

You can also set-up tabs with icons.

<Unstyled>
<Tabs skin="primary" size="sm" variant="boxed">
  <Tab id="tab1" label="Music" icon={<MusicNoteBeamed />} />
  <Tab id="tab2" label="Videos" icon={<CameraVideo />} />
</Tabs>
</Unstyled>

```jsx
<Tabs skin="primary" size="sm" variant="boxed">
  <Tab id="tab1" label="Music" icon={<MusicNoteBeamed />} >...</Tab>
  <Tab id="tab2" label="Videos" icon={<CameraVideo />}>...</Tab>
</Tabs>
```

## Full-width tabs

Stretch tab's width to the width of its parent.

<Unstyled>
<div style={{marginBottom: "2.5rem"}}>
    <Tabs skin="primary" size="sm" variant="bordered" stretch={true} stretchMode="normal">
    <Tab id="tab1" label="Music" icon={<MusicNoteBeamed />} />
    <Tab id="tab2" label="Videos" icon={<CameraVideo />} />
    </Tabs>
</div>

<div>
    <Tabs skin="primary" size="sm" variant="bordered" stretch={true} stretchMode="all">
    <Tab id="tab1" label="Music" icon={<MusicNoteBeamed />} />
    <Tab id="tab2" label="Videos" icon={<CameraVideo />} />
    </Tabs>
</div>
</Unstyled>

```jsx

<Tabs skin="primary" size="sm" variant="bordered" stretch={true} stretchMode="normal">
    <Tab id="tab1" label="Music" icon={<MusicNoteBeamed />} />
    <Tab id="tab2" label="Videos" icon={<CameraVideo />} />
</Tabs>
```

## Properties

Tabs properties:

<table>
    <tr>
        <td><strong>Property</strong></td>
        <td><strong>Type</strong></td>
        <td><strong>Default</strong></td>
        <td><strong>Description</strong></td>
    </tr>
    <tr>
        <td>label</td>
        <td>string</td>
        <td></td>
        <td>Defines the aria-label for the tab nav.</td>
    </tr>
    <tr>
        <td>skin</td>
        <td>string</td>
        <td>primary</td>
        <td>The skin/color of the tab nav. Options are: <br /> <code>primary</code> <code>success</code>  <code>warning</code>  <code>danger</code> <code>dark</code>  <code>muted</code></td>
    </tr>
    <tr>
        <td>variant</td>
        <td>string</td>
        <td>normal</td>
        <td>The variant/layout of the tab nav. Options are: <br /> <code>normal</code> <code>bordered</code>  <code>boxed</code>  <code>boxed-text</code></td>
    </tr>
    <tr>
        <td>size</td>
        <td>string</td>
        <td>md</td>
        <td>Size of the tab nav text and icon. Options are:<br /><code>tiny</code> <code>sm</code> <code>md</code> <code>lg</code></td>
    </tr>
    <tr>
        <td>stretch</td>
        <td>boolean</td>
        <td>false</td>
        <td>When true, the tabs width will stretch to match its parent</td>
    </tr>
    <tr>
        <td>stretchMode</td>
        <td>string</td>
        <td>normal</td>
        <td>Defines the stretch mode to use. Options are: <br /><code>normal</code> <code>all</code></td>
    </tr>
</table>
